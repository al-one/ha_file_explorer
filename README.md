# ha_file_explorer
在HA里使用的文件管理器

## 安装方式

请在集成中安装

```bash
# 切换到HA对应的配置目录
cd ~/.homeassistant
# 下载当前项目
git clone https://github.com.cnpmjs.org/shaonianzhentan/ha_file_explorer --depth=1
# 复制文件夹
sudo cp -r ./ha_file_explorer/custom_components/ha_file_explorer custom_components
# 删除clone的文件
sudo rm -rf ha_file_explorer

# 安装依赖
pip3 install qiniu
```
